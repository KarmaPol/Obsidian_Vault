### 규칙
- lowerCamelCase
- 동사/전치사로 시작
- 테스트 메소드는 _ 로 논리 컴포넌트를 구분
  메서드명_테스트상태_기대행위
  void isAdult_AgeLessThan18_False()
#### 자주 쓰이는 동사
- get/set
- init
  데이터 초기화
- is/has/can
  맞는지/가지고있는지/할수있는지
- create
  새로운 객체 만들고 리턴 (팩토리 메서드)
- find
  데이터 검색
- to
  객체를 다른 객체로 변환
- by
  무엇을 기준으로
