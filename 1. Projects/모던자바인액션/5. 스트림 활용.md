## 필터링
### Predicate로 필터링
```java
menu.stream()
	.filter(Dish::isVegetarian)
	.collect(toList());
```
### 고유 요소 필터링
- 고유 여부는 객체의 hashCode, equals로 결정
```java
numbers.stream()
	.filter(i -> i %2 == 0)
	.distinct()
	.collect(toList())
```

## 스트림 슬라이싱
### Predicate로 슬라이싱 TakeWhile
```java
menu.stream()
	.takeWhile(dish -> dish.getCalories() < 320)
	.collect(toList())
```
- takeWhile()로 조건이 만족할 동안만 슬라이스 -> 벗어나면 종료 
  => 무한대로 큰 배열에도 유용하다
### Dropwhile 활용
```java
menu.stream()
	.dropWhile(dish -> dish.getCalories() < 320)
	.collect(toList())
```
- 조건이 거짓인 순간 중단 후 남은 모든 요소 반환
### 스트림 축소 limit
```java
menu.stream()
	.filter(Dish::isFat)
	.limit(3)
	.collect(toList())
```
- 최대 요소 n개 반환
### 요소 건너뛰기 skip
```java
menu.stream()
	.filter(Dish::isFat)
	.skip(2)
	.collect(toList())
```
- 첫 요소 n개 제외
