# 컨플루언트 VS 아파치
- 아파치 카프카는 코어 + 커넥트 위주
- 컨플루언트 카프카는 ksql, 레지스트리가 내장
- 어드민 툴이 기본 내장

- 내부 파일 예시
	- /confluent/etc/ksqldb
	- /confluent/share/java - 모듈 별로 카프카 패키지 저장됨
